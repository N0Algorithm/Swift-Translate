# Swift Translate

A clean, responsive web translation app built with React and Vanilla CSS. Translate text and speech instantly across world languages without waiting or page reloads.

## Why We Built This

Most translation tools are cluttered with ads and require button clicks or page reloads to see results. Swift Translate offers a minimalist, distraction-free interface where translations happen instantly as you type or switch languages.

## Features

- **Instant Translation**: Real-time translation as you select languages or swap them—no submit buttons needed.
- **Dark Mode by Default**: Sleek glassmorphism UI designed for comfortable viewing.
- **Voice & Audio**: Speak into your microphone to input text and listen to native text-to-speech pronunciation.
- **Fully Responsive**: Optimized for desktops, tablets, and mobile devices with zero unnecessary scrolling.
- **History & Pinned Favorites**: Easily access recent translations or save important phrases.

## Tech Stack

- **Frontend**: React 19
- **Styling**: Vanilla CSS (Custom Design System & Glassmorphism)
- **APIs**: Web Speech API (Voice/Audio), Google Translate API (`google-translate-api-browser`), MyMemory Neural Translation API (Fallback)

## Quick Start

1. **Clone the repo:**
   ```bash
   git clone https://github.com/YOUR-USERNAME/Swift-Translate.git
   cd Swift-Translate
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run locally:**
   ```bash
   npm start
   ```

Open [http://localhost:3000](http://localhost:3000) in your browser.
